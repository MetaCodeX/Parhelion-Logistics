# Frontend Admin - Control Tower

**Stack:** Angular 18+ con Angular Material  
**Usuario:** Administrador / Gerente de Tráfico  
**Dispositivo:** PC / Laptop

## Funcionalidades

- Dashboard con KPIs operativos
- Gestión de Flotilla (Camiones tipificados)
- Gestión de Choferes
- Red Logística y Rutas Hub & Spoke
- Creación y seguimiento de Envíos
- Generación de QR para transferencia de custodia
- Documentación B2B (Carta Porte, Manifiestos)
- Modo Demo para reclutadores
