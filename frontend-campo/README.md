# Frontend Campo - App Chofer

**Stack:** React (PWA - Progressive Web App)  
**Usuario:** Chofer / Operador de Campo  
**Dispositivo:** Celular (uso con una mano)

## Funcionalidades

- Hoja de Ruta con paradas asignadas
- Confirmación de llegada a Hubs
- Escaneo de QR para toma de custodia
- Confirmación de entregas
- Captura de firma digital (POD)
- Reporte de excepciones/incidencias
